# My AI Book

Welcome to My AI Book, a comprehensive guide to AI and Robotics with a focus on ROS 2 integration.

## About this Book

This book covers various topics related to artificial intelligence and robotics, with special emphasis on connecting AI agents to humanoid robot bodies using ROS 2 as the "Robotic Nervous System".

## Getting Started

To run this documentation locally:

1. Make sure you have Node.js installed (version 18 or higher)
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

The documentation will be available at `http://localhost:3000`.

## Modules

- **ROS 2 Humanoid Integration**: Learn about ROS 2 fundamentals, Python agent integration, and humanoid modeling with URDF.
